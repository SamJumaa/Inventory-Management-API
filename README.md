# Inventory Management API

A Django REST Framework backend for managing inventory, products, purchases, sales, and stock records.

## Apps
- accounts
- catalog
- inventory
- purchases
- sales

## Features
- Product management
- Purchase management
- Sales management
- Inventory tracking
- Admin panel
- API authentication
- DRF browsable API
